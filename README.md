 Installation
 ============
 
    virtualenv --python=python2 py27tests
    source py27tests/bin/activate
    # Choose one among these
    pip install ikpdb
    pip install -e ../ikpdb
    
Launch tests
============

    cd ikpxdb_tests
    python run_tests

Development
===========

The tests can be debugged (client side) using IKPdb.
